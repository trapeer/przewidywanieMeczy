Program przewidywanie_meczy.pyw służy do wpisywania odpowiednich danych na temat meczu, po czym
program powinnien zwrócić teoretyczną ilość bramek jaką dana drużyna powinna strzelić.
Wersja z rozszerzeniem .exe daje te same możliwości jednak bez graficznego interfejsu.
Program szukanieZmiennychDlaSymulacji.exe był używany w celu znalezienia najlepszych zmiennych dla ostatecznego programu korzystając z 50 pierwszych meczy w baza_pilkarska.
Natomiast program porownywanieBaz_pilkarskich.exe służył do porównania wyników z badanej pierwszej 50 z grupą odniesienia, którą było kolejne 50 meczy z bazy_pilkarskiej.
